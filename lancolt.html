Láncolt listák

Motiváció

  • Törlés, új elem hozzáadása olcsó művelet
  • Tetszőleges méretűre bővíthető

Célok

  • Műveletek megvalósítása láncolt listákon

Lista műveletek

Beszúrás

            ListaElem *uj; 
            uj = (ListaElem*) malloc(sizeof(ListaElem)); 
            uj->kov = eleje; 
            eleje = uj;

Törlés

            lemarado = NULL; 
            mozgo = eleje; 
            while (mozgo != NULL && mozgo->adat != mit) { 
              lemarado = mozgo;     
              mozgo = mozgo->kov; 
            } 
            lemarado->kov = mozgo->kov; 
            free(mozgo);

Feladatok

  1. Írj függvényt, amely paraméterként vesz át egy számokból álló listát, és törli abból a páros számokat!
  2. Írj függvényt, amely egy lista utolsó elemét a lista elejére helyezi át!
