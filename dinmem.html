Dinamikus memóriakezelés

Motiváció

  • Nem tudjuk előre a méretet
  • Nem tudjuk kontrollálni az élettartamot
  • Nem fér el a veremben a tömb

Célok

  • Mennyi memóriát foglalunk
  • Mikor foglalunk és szabadítunk fel

Eszközök

  • A malloc() és a free() függvény
  • Realloc()

Példa használatukhoz: 

        #include <stdio.h>
        #include <stdlib.h>

        int n; 
        printf("Hány számot? "); 
        scanf("%d", &n);  
        double *tomb;
        tomb = (double*) malloc(n * sizeof(double));  

        if (tomb == NULL) {    
        printf("Nem sikerült memóriát foglalni!\n");  
        return false;   
        }  
        tomb[3] = 12; 
        free(tomb);

Feladatok

  1. Írjunk programot, amelyik bekéri, egy eltárolandó sokszögnek hány csúcsa van; utána pedig tárolja el a begépelt csúcsok koordinátáit!
  2. Írj függvényt, amelyik két sztringet vár paraméterként. Az első sztring egy feldolgozandó szöveget tartalmaz, a második pedig mindenféle karaktereket. A függvény feladata, hogy visszatérjen egy olyan dinamikusan foglalt sztringgel, amely úgy keletkezik, hogy az elsőből kihagy minden olyan karaktert, amelyik a másodikban szerepel.
